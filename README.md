# React聊天ui

1. 把项目克隆到本地
```
git clone git@github.com:jenvyxu/chat-ui.git
cd chat-ui
```

2. 预览demo
```
yarn start
```
  在浏览器输入http://localhost:3000/ 预览demo

3. 预览组件
```
yarn storybook
```
  在浏览器输入http://localhost:6006/ 预览组件
