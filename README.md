# React聊天UI

1.把项目克隆到本地
```
git clone git@github.com:jenvyxu/chat-ui.git
cd chat-ui
```

2.预览demo，在浏览器输入http://localhost:3000/
```
yarn start
```

3.预览组件，在浏览器输入http://localhost:6006/
```
yarn storybook
```
